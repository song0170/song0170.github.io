HTML Tags

a
The anchor tag is used to create links to other pages or websites using the href attribute.
form
Defines a form used to collect user input and send data using attributes such as action and method.
label
Provides a text label for an input element and improves accessibility when connected using the for attribute.
th
Defines a header cell in a table and is used to describe the content of columns or rows.
td
Represents a data cell in a table row and contains the actual table data.
